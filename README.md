
# AMARI Couture – Digital Atelier

A responsive front-end web application designed for a luxury fashion house. This project focuses on minimalist design, premium user experience, and modern CSS layout techniques.

## Key Features

* **Dynamic Appointment System:** A custom-built booking form that uses JavaScript to capture user data and trigger a personalized "Luxury Modal" confirmation.
* **Editorial Collection Grid:** An asymmetrical image gallery powered by **CSS Grid**, designed to mimic high-fashion magazine layouts.
* **Fully Responsive:** A mobile-first architecture that seamlessly transitions from a complex desktop grid to a functional mobile stack.
* **Premium UI/UX:** Features a branded color palette, custom hover animations, and fluid typography.

## Tech Stack

* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Advanced layouts using **Grid** and **Flexbox**, utilizing **CSS Variables** for easy theme management.
* **JavaScript (ES6+):** DOM manipulation for form handling and modal logic.


## Technical Highlights

* **Modal Logic:** Replaced standard browser alerts with a custom-coded modal that dynamically injects the user's name and selected time slot.
* **Grid Mastery:** Used `grid-template-areas` and span logic to create a non-linear "Lookbook" feel.
* **State Handling:** Implemented `e.preventDefault()` to handle data processing without page refreshes, ensuring a smooth "App-like" experience.
